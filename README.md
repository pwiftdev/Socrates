# $SOCRATES

Immersive memecoin landing — **Vite**, React, TypeScript, Tailwind CSS v4, Framer Motion, React Three Fiber.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Assets

Place in `public/`:

- `herosocrates.jpeg` — hero background
- `socrates_.obj.glb` — 3D bust (Philosopher section)

## Customize

Edit `src/lib/constants.ts` for contract address, social links, quotes, tokenomics, and roadmap.

## Scripts

- `npm run dev` — Vite dev server
- `npm run build` — production build to `dist/`
- `npm run preview` — preview production build
